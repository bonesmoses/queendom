# Basic Game Design Notes

* Style should be playful, whimsical, and cartoonish.
* The queen or X should:
  - take up most of the square when placed.
  - not be simple UTF8 characters.
  - be entity drawings (svg), suitable emojis, or clipart.
  - include a white border to be clearly visible on any color.
* There should be a heading including the game's name, which you can choose.
* Show remaining lives as hearts or crowns.
  - Lost lives should be a shadow or "empty" version of these icons.
* Regions should be easily distinguished from each other, so:
  - use a high contrast color palette, making sure to avoid similar colors.
  - regions should have a dark border around them to account for color-blind players.
* Make sure to avoid flickering, either from the board, the pieces, or anything else.
* Feel free to include background music if you find something suitable.
